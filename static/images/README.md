Place site images in this folder.

Recommended structure:

- `static/images/branding/site-logo.png`
- `static/images/content/`

Logo setup:

- Put your main website logo at `static/images/branding/site-logo.png`
- The public navbar and footer are already wired to use that file
- If the file is missing, the current `NORSU` badge will still appear

Common formats:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`
- `.svg`
